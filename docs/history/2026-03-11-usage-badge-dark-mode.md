## 상세 변경 내용

### 1. 다크 모드 색상 추가

Tailwind `dark:` prefix를 활용하여 다크 모드 전용 배경/보더/글자 색상 적용:

| 티어 | 배경 | 보더 | 글자 |
|------|------|------|------|
| 🟢 < 50% | `#142C1B` | `#234E30` | `#4ADE80` |
| 🟠 50-80% | `#322014` | `#59361C` | `#FB923C` |
| 🔴 >= 80% | `#2C1414` | `#4E2323` | `#F87171` |

- 프로젝트의 `darkMode: ['class']` 설정에 맞는 표준 방식 사용
- 라이트 모드 파스텔 색상은 기존 유지

## 테스트 방법

1. 다크 모드에서 top bar 사용량 뱃지 확인
2. 라이트 모드 전환 시 파스텔 색상 유지 확인
3. 3개 티어(< 50%, 50-80%, >= 80%) 각각 시각 확인
